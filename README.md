# Automated Software Project Estimator Using FPA and COCOMO
Automated Software Project Estimator tool created by students of 7th Semester BTech CSE for partial fulfillment of the requirement of Bachelor of Technology in Computer Science and Engineering. This project utilised Function Point Analysis and COCOMO estimation models to achieve a rough estimate of the cost and time involved in a software project.

### Developers: [Tabish Perwaiz (BTech CSE)](https://github.com/tabishperwaiz21), [Mirza Mohammad, BTech CSCE](https://github.com/Mirza1301-ops)

## Running This Project

Clone this repo.

Install Dependencies using the ```requirements.txt```.

Terminal 1

```cd backend```
<br>
```python -m uvicorn main:app --reload```

Terminal 2

```cd project-estimator```
<br>
```npm run dev```
